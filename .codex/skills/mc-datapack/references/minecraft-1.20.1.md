# Minecraft Java 1.20.1 Datapack Notes

- `pack.mcmeta` uses `pack.pack_format` value `15`.
- Function tag paths:
  - `data/minecraft/tags/functions/load.json`
  - `data/minecraft/tags/functions/tick.json`
- Function files use `.mcfunction`.
- A function at `data/example/functions/foo/bar.mcfunction` is called with `/function example:foo/bar`.
